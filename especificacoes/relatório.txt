Inicialmente,considerou-se usar outro software que não o GeoServer. Google Maps foi escolhido, juntamente com sua API de Javascript. Para que os desenhos sejam executados, é necessário que um JSON com a coordenadas em latitude e longitude seja passado pela API. Os bancos de dados de geoprocessamente não tem as coordenadas em ll(latitude e longitude), apenas em UTM(DATUM WGS 84 FUSO 22 ZONA H). Para a conversão, é necessária a implementação de um código específico, para assim poder criar o JSON com as coordenadas.
